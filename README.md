## Problem Statement
India faces a shortage of judicial officers and a growing case backlog. One major reason is the lack of early awareness and guidance about the judicial career path, especially among students from non-urban and non-English backgrounds.

## Solution Overview
Nyaya-Uday is a lightweight, mobile-first web application that helps students discover the judicial career path early. It provides a clear, state-specific roadmap and allows users to experience judicial thinking through simple case simulations.

## Key Features (MVP)
- State-specific judicial career roadmap
- Simple and jargon-free guidance
- Junior Judge simulation
- Judicial Aptitude Score
- Mobile-first design
